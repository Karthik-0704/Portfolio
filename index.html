Interpretable Text-to-SQL with RAG + TRC

10/2025 – 12/2025

Built a retrieval-augmented Text-to-SQL system for SParC (multi-turn) and Spider using an interpretable TRC intermediate representation with a hybrid LLaMA 3.1 8B (Groq) + fine-tuned Flan-T5 pipeline.

  • Schema-aware vector retrieval (tables/columns) with semantic grounding and schema expansion (PK/FK).
  • Integrated Neo4j graph DB + MySQL schema ingestion via INFORMATION_SCHEMA with live sync.
  • Deterministic TRC→SQL compiler + optional LLM SQL generator for safe read-only evaluation.
